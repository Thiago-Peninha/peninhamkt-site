🔗 CONEXÃO COM GITHUB

O conteúdo do site é salvo diretamente no seu repositório GitHub.
Para isso funcionar, você precisa configurar um Personal Access Token com permissão de escrita.

Verificando...
📁 Repositório: Thiago-Peninha/peninhamkt-site
📄 Arquivo de dados: content.json

Como gerar o token:

  1. Acesse github.com/settings/tokens/new
  2. Em Note, escreva: Peninha Site CMS
  3. Em Expiration, selecione No expiration
  4. Marque a permissão: ✅ repo (acesso completo)
  5. Clique em Generate token e copie o token gerado
  6. Cole abaixo e clique em Salvar Token

🔒 Segurança: O token fica salvo apenas neste navegador (localStorage). Nunca é enviado a terceiros — apenas à API oficial do GitHub. Cada computador que usar o admin precisa configurar o token uma vez.
